# Canvas Design — Instrucciones de Ejecución

## El proceso tiene dos pasos

### PASO 1 — Filosofía de diseño

Antes de generar, definir el concepto visual:
- ¿Qué emoción comunica esta pieza?
- ¿Cuál es la tensión visual principal? (grande/pequeño, negro/blanco, foto/tipo)
- ¿Dónde está el elemento rojo y qué rol cumple?
- ¿Qué jerarquía tienen los textos?

### PASO 2 — Ejecución en canvas

**Principios críticos de ejecución:**

- Generar trabajo que parezca haber tomado horas. Meticulosamente elaborado.
- Cada espaciado, cada tamaño tipográfico, cada alineación — revisado con precisión.
- Nada se superpone. Nada se sale del canvas. Todo tiene margen y aire.
- El resultado tiene que poder mostrarse como prueba de expertise.

**Tipografía como diseño:**
- Los títulos no son solo texto — son elementos arquitectónicos
- Usar contrastes extremos de tamaño (ej: 80pt con 10pt en la misma pieza)
- Las palabras pueden romperse, apilarse, sangrar — intencionalmente

**Color:**
- Paleta limitada = más impacto
- El rojo dirige la mirada — usarlo estratégicamente, no decorativamente
- El negro y blanco hacen el trabajo pesado

**Fotografía (cuando aplica):**
- Convertir a B&W primero
- Aplicar overlay rojo semitransparente (40–60% opacidad)
- Integrar con la tipografía — que se superpongan, no que coexistan separadas

**Detalles finales:**
- Siempre incluir logo + web en piezas externas
- Revisar que nada esté cortado ni fuera del canvas
- Segunda pasada de refinamiento antes de entregar

**Output:** `.pdf` para multi-página, `.png` para pieza individual.

## Para múltiples páginas

Cada página es única pero parte de una familia visual coherente.
Variar composición entre páginas mientras se mantiene la paleta y sistema tipográfico.
